CE marking of machinery,
built as a model.

A free, open-source tool for the Machinery Regulation (EU) 2023/1230. Build the conformity work once as a model of typed entities, and generate the hazard lists, requirement specifications, and traceability from it.

The interface: a navigator, an editor, and the relationships of the selected entity.

What it is

CE marking produces a technical file: a risk assessment, a list of applicable essential requirements, the standards applied, the measures taken, and the traceability between them. Done in documents, each of these is written and maintained separately, and they drift apart.

openconformity treats the same work as a single model. Every hazard, requirement, and measure exists exactly once, with typed relationships to the others. The documents become views of that model rather than the model itself.

It runs entirely in the browser. There is no account, nothing to install, and no server: the project is a single file on your own machine, and nothing you enter leaves it.

The model

Entities are typed, and so are the relationships between them. A relationship carries meaning, which is what makes a chain like this readable in both directions, and what lets a change at one end surface at the other.

SE-001Packaging machine
decomposes into
SE-002Infeed unit
exhibits
HZ-003Crushing at the infeed rollers
mitigated by
RRM-004Fixed guard

The entity types available, and which relationships may connect them, are fixed by a metamodel built into the tool. It is what keeps a model consistent enough to generate artefacts from.

Status

In development, and not yet usable. The demo shows the interface with fixed example content; the tool behind it is being built.

The work is public. The requirements specification, the decision log, and the source are all in the repository. Progress is posted on LinkedIn.

openconformity produces artefacts for the user to review and use. It does not assert conformity. The manufacturer remains responsible for the conformity assessment and for the compliance of the product.
